# UtopiaHire Frontend

React + TypeScript + Vite frontend for the UtopiaHire platform.

## Setup

### Install Dependencies
```bash
npm install
```

### Start Development Server
```bash
npm run dev
```

The frontend will be available at http://localhost:5173

Backend API URL is configured in `.env`:
```

```

```bash

```
